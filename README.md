# 汕博雅口腔官方网站

## 项目简介

这是一个为汕博雅口腔诊所设计的现代化官方网站，采用响应式设计，支持桌面端和移动端访问。

## 技术栈

- **HTML5** - 语义化标签
- **CSS3** - 现代化样式，使用Flexbox和Grid布局
- **JavaScript** - 交互功能
- **Font Awesome** - 图标库

## 项目结构

```
/
├── index.html          # 首页
├── about.html          # 关于我们
├── team.html           # 医生团队
├── services.html       # 诊疗项目
├── process.html        # 就诊流程
├── faq.html            # 常见问题
├── contact.html        # 联系我们
├── styles.css          # 样式文件
├── script.js           # JavaScript文件
└── README.md           # 项目说明
```

## 功能特点

### 1. 响应式设计
- 支持桌面、平板和移动设备
- 移动端友好的导航栏
- 自适应网格布局

### 2. 核心页面
- **首页**：诊所介绍、特色服务、联系方式
- **关于我们**：使命、愿景、核心价值观
- **医生团队**：林冠峰医生团队介绍
- **诊疗项目**：牙齿矫正、种植牙、儿童牙科、综合治疗
- **就诊流程**：详细的就诊步骤
- **FAQ**：常见问题解答（可折叠）
- **联系我们**：联系方式和预约信息

### 3. 交互功能
- 移动端导航栏切换
- 平滑滚动
- 返回顶部按钮
- 页面滚动进度条
- FAQ折叠展开功能
- 悬停动画效果

### 4. SEO优化
- 结构化数据（Schema.org）
- Meta标签优化
- 语义化HTML标签

## 本地预览

### 方法1：使用Python
```bash
python3 -m http.server 8000
```
然后在浏览器中访问：http://localhost:8000

### 方法2：使用Node.js
```bash
npx http-server
```

### 方法3：使用VS Code
安装Live Server插件，右键点击index.html选择"Open with Live Server"
