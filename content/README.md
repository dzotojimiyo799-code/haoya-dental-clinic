## 使用指南

### 发布到官网
1. 将`week1/article-*.md`文件复制到官网文章后台
2. 选择合适的分类（牙齿矫正、种植牙、儿童牙科等）
3. 设置发布时间，建议每周发布2篇

### 发布到小红书
1. 将`week1/xiaohongshu-*.md`文件内容复制到小红书发布框
2. 添加话题标签（已包含在文件中）
3. 上传相关图片（诊所环境、医生照片、治疗案例）
4. 设置发布时间，建议每周发布3篇

### 发布到知乎
1. 在知乎搜索相关问题
2. 将`week1/zhihu-*.md`文件内容作为回答
3. 添加相关图片和视频（如有）
4. 优化回答格式，提高可读性

### 发布到百家号
1. 将科普文章和FAQ发布到百家号
2. 选择合适的分类
3. 添加相关图片
4. 设置发布时间

### 发布到短视频平台
1. 根据`week1/video-*.md`脚本拍摄视频
2. 按照脚本内容制作
3. 添加字幕和背景音乐
4. 发布到抖音、快手、视频号等平台

### 优化地图平台
1. 登录百度地图、高德地图、腾讯地图、大众点评
2. 编辑诊所信息
3. 复制`maps/platform-content.md`中对应平台的内容
4. 上传诊所外景、诊室、医生等照片
5. 确认联系方式和营业时间准确
