## 🌟 方案一：GitHub Pages（最推荐）

### 优势
✅ **完全免费** - 无需任何费用
✅ **自动化部署** - 推送代码自动更新
✅ **HTTPS证书** - 自动配置SSL证书
✅ **自定义域名** - 支持绑定自己的域名
✅ **全球CDN** - 访问速度快
✅ **版本控制** - Git管理，便于回滚

### 部署步骤

#### 1. 注册GitHub账号
- 访问：https://github.com
- 点击 "Sign up" 注册账号
- 完成邮箱验证

#### 2. 创建仓库
- 登录GitHub
- 点击右上角 "+" → "New repository"
- 仓库名称：`boya-dental`（建议）
- 勾选 "Public"（公开）
- 勾选 "Initialize this repository with a README"
- 点击 "Create repository"

#### 3. 上传网站文件

**方法一：使用GitHub网页界面上传**
1. 进入刚创建的仓库
2. 点击 "Add file" → "Upload files"
3. 将以下文件拖拽上传：
   - `index.html`
   - `about.html`
   - `team.html`
   - `services.html`
   - `process.html`
   - `faq.html`
   - `contact.html`
   - `articles.html`
   - `articles.css`
   - `styles.css`
   - `script.js`
   - `articles/` 文件夹及其内容
4. 在 "Commit changes" 输入提交信息
5. 点击 "Commit changes"

**方法二：使用Git命令行（推荐）**
```bash
# 1. 进入网站目录
cd /Users/mac/WorkBuddy/20260313094845

# 2. 初始化Git仓库
git init

# 3. 添加所有文件
git add .

# 4. 提交文件
git commit -m "首次提交汕博雅口腔网站"

# 5. 关联GitHub仓库
git remote add origin https://github.com/你的用户名/boya-dental.git

# 6. 推送到GitHub
git branch -M main
git push -u origin main
```

#### 4. 启用GitHub Pages
1. 进入仓库页面
2. 点击 "Settings"（设置）
3. 左侧菜单找到 "Pages"
4. 在 "Source" 下选择：
   - Branch: `main`
   - Folder: `/ (root)`
5. 点击 "Save"
6. 等待1-2分钟，页面刷新后会显示网站地址：
   ```
   https://你的用户名.github.io/boya-dental
   ```

#### 5. 访问网站
打开浏览器访问：
```
https://你的用户名.github.io/boya-dental
```

### 自定义域名（可选）

#### 1. 购买域名
推荐域名商：
- 阿里云：wanwang.aliyun.com
- 腾讯云：dnspod.cn
- Cloudflare：https://www.cloudflare.com

推荐域名：
- `boya-dental.com`
- `stboyadental.com`
- `shantou-boya.com`

#### 2. 配置DNS解析
1. 在域名商控制台添加CNAME记录
   - 主机记录：`@` 或 `www`
   - 记录值：`你的用户名.github.io`
   - TTL：10分钟

#### 3. 在GitHub绑定域名
1. 进入仓库 "Settings" → "Pages"
2. 在 "Custom domain" 输入你的域名
3. 点击 "Save"
4. DNS自动验证后，域名生效

### 文件结构要求
GitHub Pages支持以下文件结构：
```
boya-dental/
├── index.html           # 首页（必需）
├── about.html
├── articles.html
├── articles/
│   ├── article-1-orthodontics-price.html
│   └── article-2-implants-guide.html
├── styles.css
├── articles.css
├── script.js
└── ...
```

### 更新网站
修改文件后，只需推送代码：
```bash
git add .
git commit -m "更新文章内容"
git push
```
GitHub Pages会自动重新部署网站。

## 🚀 方案二：Netlify（最简单）

### 优势
✅ **拖拽部署** - 最简单的部署方式
✅ **自动HTTPS** - 免费SSL证书
✅ **自定义域名** - 免费绑定
✅ **全球CDN** - 访问速度极快
✅ **表单功能** - 免费联系表单
✅ **无限制流量** - 免费套餐

### 部署步骤

#### 1. 注册Netlify账号
- 访问：https://www.netlify.com
- 点击 "Sign up"
- 使用邮箱注册

#### 2. 拖拽部署
1. 登录Netlify
2. 点击 "Add new site" → "Deploy manually"
3. 将网站文件夹拖拽到虚线框内
4. 等待部署完成（约30秒）
5. 获得网站地址：`https://随机名称.netlify.app`

#### 3. 更改网站名称
1. 进入 "Site settings"
2. 点击 "Change site name"
3. 输入：`boya-dental`
4. 访问：`https://boya-dental.netlify.app`

#### 4. 绑定自定义域名
1. 进入 "Domain management"
2. 点击 "Add custom domain"
3. 输入你的域名
4. 配置DNS CNAME记录到 `boya-dental.netlify.app`

### 持续部署（可选）
可以将网站文件夹连接到GitHub，每次推送代码自动部署。

## ☁️ 方案五：阿里云OSS（对象存储）

### 优势
✅ **国内速度快** - 阿里云CDN
✅ **成本低** - 按量计费
✅ **HTTPS** - 免费SSL证书
✅ **自定义域名** - 支持

### 费用估算
- 存储费用：约 0.12元/GB/月
- 流量费用：约 0.5元/GB（首年可能有优惠）
- 网站预估：每月约 5-20元

### 部署步骤

#### 1. 开通OSS服务
1. 登录阿里云控制台
2. 搜索 "对象存储 OSS"
3. 点击 "立即开通"

#### 2. 创建Bucket
1. 进入OSS控制台
2. 点击 "创建Bucket"
3. 配置：
   - Bucket名称：`boya-dental`
   - 地域：选择华南1（深圳）- 距离汕头近
   - 存储类型：标准存储
   - 访问权限：公共读

#### 3. 上传文件
1. 进入Bucket
2. 点击 "上传文件"
3. 上传所有网站文件

#### 4. 配置静态网站
1. 进入Bucket设置
2. 点击 "静态页面"
3. 配置：
   - 默认首页：`index.html`
   - 默认404页：`404.html`
4. 点击 "保存"

#### 5. 绑定自定义域名
1. 进入域名管理
2. 点击 "绑定域名"
3. 输入域名
4. 配置DNS CNAME记录

#### 6. 配置HTTPS（推荐）
1. 进入Bucket "传输管理" → "域名管理"
2. 点击 "证书托管"
3. 上传或申请免费SSL证书
4. 开启HTTPS

## 🔧 技术支持

### 常见问题

**Q: GitHub Pages访问慢怎么办？**
A: 可以考虑使用阿里云OSS或Netlify，国内访问会更快。

**Q: 如何更新网站内容？**
A: GitHub Pages和Netlify都支持Git推送自动更新，只需修改文件并推送即可。

**Q: 需要购买域名吗？**
A: 不强制，但购买自定义域名更专业。可以使用平台的免费域名。

**Q: HTTPS证书需要付费吗？**
A: 以上方案都提供免费HTTPS证书。

**Q: 可以托管动态网站吗？**
A: 以上方案适合静态网站（HTML/CSS/JS）。如果需要数据库、用户登录等功能，需要云服务器。
