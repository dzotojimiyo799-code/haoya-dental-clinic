### 方法三：使用 SSH 密钥

#### 步骤 1：生成 SSH 密钥

```bash
ssh-keygen -t ed25519 -C "dzotojimiyo799-code@github.com"
```

按回车使用默认值（不需要输入密码）

#### 步骤 2：添加 SSH 密钥到 GitHub

1. 复制公钥内容：

   https://github.com/settings/ssh/new

3. 粘贴公钥内容，点击 `Add SSH key`

## ⚠️ 常见问题

### Q1: 推送时提示 "Authentication failed"？

**解决方案：**
- 检查 Token 是否正确复制
- 确认 Token 的有效期
- 重新生成新的 Token

### Q2: Token 在哪里可以找到？

**解决方案：**
- Token 只在创建时显示一次
- 如果丢失，需要重新生成：https://github.com/settings/tokens

### Q3: 如何保存 Token 以便下次使用？

**解决方案：**
使用 Git Credential Helper 自动保存凭证：

```bash
git config --global credential.helper store
```

下次推送时只需输入一次密码，之后就会自动保存。
