## ⚠️ 常见问题

### Q1: Pages 设置找不到怎么办？

**解决方案：**
- 确保您已登录 GitHub
- 确认您有该仓库的写入权限
- 刷新页面重试

### Q2: 保存后看不到部署状态？

**解决方案：**
- 等待 30 秒刷新页面
- 检查仓库的 Actions 标签
- 查看是否有部署错误

### Q3: 部署失败怎么办？

**解决方案：**
1. 查看 Actions 日志
2. 检查文件结构
3. 确保 index.html 在根目录
4. 重新触发部署

### Q4: 网站访问 404？

**可能原因：**
- GitHub Pages 未启用
- 部署尚未完成
- URL 错误

**解决方案：**
- 确认 Pages 设置正确
- 等待 2-3 分钟后刷新
- 检查 URL 拼写
